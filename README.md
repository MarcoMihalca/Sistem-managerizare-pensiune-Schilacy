Sistem de Gestiune Hotelieră



Proiect pentru materia "Ingineria Programelor", realizat de studentii Mihalca Marco Alexis Dimitri si Popenciuc Andrei, de la Calculatoare III, în legătură cu crearea unei aplicații pentru managerizarea, eficientizarea și modernizarea Pensiunii Schilacy.
Aplicatia manager pentru "Pensiunea Schilacy" este solutia software dezvoltata pentru a aduce aceasta pensiune in era tehnologiei.
Ca obiective principale, avem digitalizarea si reducerea de timp a check-in-ului si a check-out-ului.



!! IMPORTANT !! Documentația acestui proiect, acompaniată de către instrucțiunile de rulare și foaia cu cerințe, semnată, se afla în folderul "./docs/Documente".
