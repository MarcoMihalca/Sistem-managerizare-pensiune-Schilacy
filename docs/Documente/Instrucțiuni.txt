Pentru a rula aplicatia, în folderul principal unde se afla fișierul "manage.py", in terminalul editorului de cod, precum Visual Studio Code, se va introduce urmatoarea comanda 'python manage.py runserver' pentru a rula aplicatia pe un server local, iar mai apoi se va deschide browser-ul ales, introducand adresa 'localhost:8000', respectiv 'http://127.0.0.1:8000', in adresa de cautare. Pentru a accesa pagina de admin, se va adauga '/admin' la adresele specificate anterior. Pentru a se loga se vor folosi urmatoarele credentiale user = 'admin', parola = '1234'. Înainte de prima rulare, asigurati-vă că aveți instalate librăriile necesare folosind comanda 'pip install -r requirements.txt'.
